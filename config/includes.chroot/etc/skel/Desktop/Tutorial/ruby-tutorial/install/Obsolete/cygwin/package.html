cygwinにパッケージでインストールする

ここではcygwinにruby dcl関係のインストールを簡単に行う方法をのせます。 ここに載っている方法でインストールすると

  • gnome2関係
  • ruby-gnome2関係
  • netcdf
  • ruby-netcdf
  • dcl(C)+gtk2パッチ
  • ruby-dcl+gtk2パッチ
  • narray
  • narray_miss
  • misc
  • gphys
  • gave
がインストールされます。 また個別にインストールすることもできますが、その場合は依存関係に注意して下さい。

インストール

  • cygwinのsetup.exeを実行します
    • お手元の環境に合わせてサーバーの選択まで進んでください
  • サーバーの選択
    • 適当なサーバーを選択
    http://web.sfc.keio.ac.jp/~s01397ms/cygwin/ を入力し、Add を押す
    ftp://www.gfd-dennou.org/arch/ruby/products/cygwin/ を入力し、Add を押す
    次へ をクリック
  • パッケージの選択
    • GNOME2, Dennou のカテゴリーの Default を数回クリックし、Install に変える
    以下を追加(括弧内はカテゴリー名)
    • XFree86-bin (XFree86)
    • ruby (Interpreters or Develop)
    • gtk2-win32 (Libs)

    カテゴリーパッケージ
    GNOME2ALL
    DennouALL
    XFree86XFree86-bin
    Interpreters or Developruby
    Libsgtk2-win32

rubyのための準備

rubyのサイトパスは /usr/lib/ruby/site_ruby ですが、 上記のパッケージでは /usr/local/lib/ruby/site_ruby にインストールされるので
リンクを張ります
%cd /usr/lib/ruby
%mv site_ruby site_ruby.bak
%ln -s /usr/local/lib/ruby/site_ruby .

これでインストールが終わりました。
