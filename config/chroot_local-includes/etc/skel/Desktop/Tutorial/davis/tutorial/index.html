電脳Rubyチュートリアル

チュートリアル(全般)

電脳Ruby諸資源を使ってデータ解析と可視化を行うためのチュートリアル。 Rubyの全くの初心者が、なにもインストールしてないところからはじめられる親切なメニューです。

  • いんすとーる編

  • ruby 基礎編
    (Rubyの初歩 / Ruby対話利用 / 多次元数値配列扱いと可視化の初歩 / NetCDF)

  • ちょっと進んだ使いかた編

チュートリアル(個別)

  • GPhysのチュートリアル
    (GPhysは物理量データを扱う統合解析可視化ライブラリーです．)

    • GPhysコマンドチュートリアル
      (GPhys付属のアプリケーションコマンド群の使い方)

    • GPhys::EP_Flux のチュートリアル

  • RDocによる自動ドキュメント生成
    (RDocはRubyソースから自動でhtmlドキュメントを生成します. コーディング時にRDocスタイルのコメントをつけていくようにすると, 自ずと見やすいドキュメントが作れます.)

  • 気象データ解析ライブラリの開発のチュートリアル

  • 拡張ライブラリー作成
    • Ruby 拡張ライブラリー作成チュートリアル
      (for C; 直書きとSWIG利用の両方)

    • SWIGによる拡張ライブラリー自動生成のチュートリアル
      (for Fortran, C & C++. ちょっと古いです)

  • Ruby/GSL を用いたデータ解析スクリプトの紹介
    (Ruby/GSLは科学計算用ライブラリー GSL (GNU Scientific Library) のRubyラッパーです)

  • Ruby-VTK Tutorial
    (Ruby-VTK は３次元描画ライブラリー VTK (Visualization ToolKit) のRubyラッパーです)

古いチュートリアル

  • もっと進んだ使いかた編 (この中のGPhysチュートリアルは古いです。)

参考

  • ドキュメンテーションリンク集: 電脳 Ruby 生活を快適に送るための、ライブラリーのマニュアルなどのリンク集。 それぞれのホームページへのリンクもあります。

お薦めの文献 (ちょっと古いです)

  • Rubyオフィシャルホームページ オンラインマニュアルが便利。

  • 「Rubyデスクトップレファレンス」 まつもと著 (オライリージャパン) オンラインマニュアルよりは本が見やすいという場合はこれ。

  • 「プログラミングRuby」トーマス＆ハント著(ピアソン・エデュケーション)
    定評ある著者による好著。入門用にお薦め。Ruby 1.6ベース。
    原版(英語)は "Programming Ruby" by Thomas and Hunt (Addison-Wesley)

  • 「オブジェクト指向スクリプト言語Ruby」まつもと・石塚著(アスキー出版局)
    Rubyの能力をフルに活かしたい人に必携。 拡張ライブラリーの作りかたの説明もあります。 ただし、Rubyをはじめて学ぶひとには敷居が高いかもしれません。索引は不備。 Ruby 1.4ベース。

  • 「dRubyによる分散オブジェクトプログラミング」関著(アスキー出版局)
    その名の通り！

  • 「eRuby テキスト埋め込み型 Ruby」高井著(毎日コミュニケーションズ)
    htmlページを書くときに役に立ちます。

  • 「オブジェクト指向における再利用のためのデザインパターン」 Gammaら著(ソフトバンク)
    より使い回しの効くプログラムが書けるようになりたい方は、一度目を通しておくとよいでしょう。

  • 「オブジェクト指向入門」Meyer著 (アスキー)
    第1版の日本語訳です。 第2版は英語のみ： "Object-Oriented Software Construction (2nd Edition)" by Meyer (Prentice Hall) (どちらを読んでもいいでしょう). オブジェクト指向の必要性と効能が懇切丁寧に書かれてておすすめ。
