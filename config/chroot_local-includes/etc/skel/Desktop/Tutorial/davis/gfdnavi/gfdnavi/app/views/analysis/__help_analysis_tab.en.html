Analysis tab

  1. Choose one or more variables with the check-boxes in the "Variables" window at the top. Kinds of functions you can choose below depends on the number of variables you choose. For instance, you can "add" variables if you choose two.
  2. Click this tab, and select a function with the "Function" field. You can add your own one if you have an account.
  3. Press the "analyze" button above (next to the "Variables" window. This button is shown only when it is ready).

OPTIONS:

  • Specify a subset with the "Dimensions" window above. The function is applied only to the subset. Recommended if the data size is big.
  • "Number of input variables" field: You can use it like this: If you choose two or more variables in the "Variables" window, you can still use functions for single variables (such as "mean"), if you choose "1" here. Then, the function is applied for each variable.
  • "Arguments for the function" text window: Some of the function require arguments. In that case, this window appears and shows its default value, which you can change by typing in something else.
