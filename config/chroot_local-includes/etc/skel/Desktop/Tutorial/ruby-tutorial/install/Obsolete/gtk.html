GTKに関する情報

GTKについて

現在、GTKには 1系(1.*.*)と2系(2.*.*)の大きく分て二つのバージョンの系統があり、 これらは互換性がありません。 インストールする際には注意が必要です。

DCL with GTK

dcl で GTK を利用するためには DCL(C) と ruby-dcl にパッチをあててコンパイルする必要があります。 ここを参考にインストールしてください。
上記のパッチは GTK1、GTK2両方に対応しています。

DCL(C)のコンパイル時に configure スクリプトがあなたの環境をチェックし、 GTK2が見つかればGTK2を、 GTK2が見つからずにGTK1がみつかればGTK1を、 GTK1も見つからなければGTKなしでコンパイルします。
インストール後
%cdclconfig --gtkversion
とすれば使用されたGTKのバージョンが分かります (2->GTK2, 1->GKT1, no->GTKなし)。

ruby-dclのコンパイル時は自動的に DCL(C) と同じバージョンの GTKを使います。 インストール後
%ruby -r numru/dcl -e "p NumRu::DCL.gtk_version"
とすれば使用されたGTKのバージョンが分かります ([2, 2, 4] -> gtk-2.2.4 )。
また ruby-dcl はおなじGTKバージョンの ruby-gtk(ruby-gnome) or ruby-gtk2(ruby-gnome2) を必要とします。
GTK2がインストールされている環境では ruby-gtk2を、 GTK1なら ruby-gtk をインストールするようにしましょう
