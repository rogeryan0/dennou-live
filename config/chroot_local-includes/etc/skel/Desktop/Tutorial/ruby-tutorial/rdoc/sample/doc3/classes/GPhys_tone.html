Public Instance methods

tone(itr=1)

描画メソッド

色塗り図を描画するメソッド. 等値線図のみを描画したい場合は GPhys_cont#cont を利用してください.

itr :描画する際の地図投影法を指定します. 数値を与えてください. デフォルトは 1 になっています. 番号と投影法の関係に関しては www.gfd-dennou.org/library/dcl/dcl-f90/doc/term/2d.htm を参照ください
返り値:常に true が返ります.
