密度流 (重力流)


  • 仕切りで水平に並べて置かれた密度の異なる 2 つの流体が, 仕切りをとった後にどのように流れるかを観察する室内実検を模したもの. 赤色が密度の大きい(重い)流体, 青色が密度の小さな(軽い)流体を表している.
  • 仕切りを外した後は, 重い流体が水層の底を這うように流れて行く. 先端部の明瞭な形状に比較して, 後部は流れるうちに乱れていく.
